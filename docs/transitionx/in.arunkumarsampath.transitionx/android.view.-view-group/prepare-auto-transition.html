prepareAutoTransition

inline fun ViewGroup.prepareAutoTransition(cancel: Boolean = false, autoTransitionBuilder: AutoTransitionBuilder.() -> Unit = {}): Unit

Schedules a call to TransitionManager.beginDelayedTransition with receiver ViewGroup as the scene root. autoTransitionBuilder uses a AutoTransition instance internally and its properties can be modified by using available methods from AutoTransitionBuilder.

Since AutoTransition is used, View bounds changes are animated using ChangeBounds and disappear/appear are animated using Fade

Parameters

cancel - if true, ends all previous transitions before starting this transition.

autoTransitionBuilder - Builder to construct transition instance for use with TransitionManager
